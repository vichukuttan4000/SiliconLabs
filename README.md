# Silicon Labs Projects

This repository contains three embedded projects for Silicon Labs SoC devices.

## Projects

1. **aiml_soc_voice_control_light_siwg917_baremetal2** - Voice-controlled light using TensorFlow Lite for keyword spotting
2. **aiml_soc_voice_control_light_siwg917_baremetal1** - Alternative voice control implementation
3. **sl_si91x_veml6035** - VEML6035 ambient light sensor example

Each project folder contains its own README with detailed instructions.

## Getting Started

Navigate to any project folder and refer to its README for build and deployment instructions.
